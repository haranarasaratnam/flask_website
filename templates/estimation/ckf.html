Cubature Kalman Filters (CKF)

Introduction

  • In deriving the CKF, we trade off global optimality for computational traceability and robustness
  • Assume that the joint state-innovations density given the past measurement history to be Gaussian
  • Under the above assumption, the new problem is to compute integrals whose integrands are of the form: \[\text{nonlinear function x Gaussian}\]
  • The logical choice is to apply the monomial based cubature rule to compute them numerically

Page 1

Numerical Integration Using Spherical-Radial Cubature Rule

  • As shown previously, we are interested in numerically computing the integral of the form: \[ I(\mathbf{f}) = \int_{R^{n_{x}}}\mathbf{f}(\mathbf{x})\exp(-\mathbf{x}^{T}\mathbf{x})d\mathbf{x} \tag{1} \]
  • As a first step, we transform \(\mathbf{I}()\) in the cartesian co-ordinate into the spherical-radial coordinate
  • This is equivalent to finding the solution to the radial integral \[ \mathbf{I}_{r} = \int_{0}^{\infty}S(r)r^{n-1}\exp(-r^{2})dr \tag{2} \] where \(S(r)\) is defined by another integral, which we call the spherical integral: \[ S(r) = \int_{U_{n}}\mathbf{f}(r\mathbf{y})d\sigma(\mathbf{y}) \tag{3} \] where \(\sigma()\) is the spherical surface measure on the region \[ U_{n} = \{\mathbf{y} \in R^{n}|\mathbf{y}^{T}\mathbf{y} = 1\} \]

Page 2

Spherical-Radial Cubature Rule (Continued)

  • We fix the degree of the target cubature rule to be 3 for numerical reasons.
  • We apply the third-degree monomial based cubature rule for the spherical integral in (3) as follows: \[ \int_{U_{n}}\mathbf{f}(r\mathbf{s})d\sigma(\mathbf{s}) \approx\sum_{i=1}^{2n}\omega_{s}\mathbf{f}[u]_{i} \tag{4} \]
  • We apply the first-degree quadrature rule for the radial integral in (2): \[ \int_{0}^{\infty}f(r)r^{n-1}\exp(-r^{-2})dr\approx \omega_{r}f(r) \tag{5} \]
  • Substituting (4) and (5) back into (1) yields the third-degree spherical-radial cubature rule: \[ \int_{R^{n}}\mathbf{f}(\mathbf{x})\exp(-\mathbf{x}^{T}\mathbf{x})d\mathbf{x} \approx \sum_{i=1}^{2n}\omega_{n}\omega_{r}\mathbf{f}([\mathbf{x}_{r}u]_{i}) \tag{6} \]

Page 3

Spherical-Radial Cubature Rule (Continued)

  • We can extend the above cubature rule to compute an \(n\)-dimensional Gaussian-weighted integral numerically: \[ \int_{R^{n}}\mathbf{f}(\mathbf{x})\mathcal{N}(\mathbf{x};\mathbf{\mu}, \mathbf{\Sigma})d\mathbf{x} \approx \frac{1}{2n}\sum_{i=1}^{2n}\mathbf{f}(\mathbf{\mu}+\sqrt{\mathbf{\Sigma}}\xi_{i}), \tag{7} \] where \(\mathbf{\Sigma}= \sqrt{\mathbf{\Sigma}}\sqrt{\mathbf{\Sigma}}^{T}\) and the cubature points \[ \xi_{i} = \begin{cases} \sqrt{n}\mathbf{e}_{i }, i=1, 2,\ldots n\\ -\sqrt{n}\mathbf{e}_{i }, i=n+1, n+2, \ldots 2n \end{cases} \tag{8} \]
  • The resulting filter now operates only on the numerically computed means and covariances of the (approximate) Gaussian conditional probability densities encountered in the time and measurement updates as described next.

Page 4

Time-Update

  • Assume at time \(k\) that the posterior density, \[ p(\mathbf{x}_{k-1}|D_{k-1}) = \mathcal{N}(\hat{\mathbf{x}}_{k-1|k-1}, \mathbf{P}_{k-1|k-1}) \tag{9} \] is known. Factorize \[ \mathbf{P}_{k-1|k-1} = S_{k-1|k-1}S_{k-1|k-1}^{T} \tag{10} \]
  • Evaluate the cubature points: (\(i=1, 2,\ldots m, \text{ where } m=2n_{x}\)) \[ X_{i, k-1|k-1} = S_{k-1|k-1}\xi_{i} + \hat{\mathbf{x}}_{k-1|k-1} \tag{11} \]
  • Evaluate the propagated cubature points \((i=1,2,\ldots m)\): \[ X^{\ast}_{i, k|k-1} = \mathbf{f}(X_{i,k-1|k-1}, \mathbf{u}_{k}) \tag{12} \]
  • Estimate the predicted state \[ \hat{\mathbf{x}}_{k|k-1} = \frac{1}{m}\sum_{i=1}^{m}X^{\ast}_{i, k|k-1} \tag{13} \]
  • Estimate the predicted error covariance \[ \mathbf{P}_{k|k-1} = \frac{1}{m}\sum_{i=1}^{m}X^{\ast}_{i, k|k-1}X^{\ast T}_{i, k|k-1} - \hat{\mathbf{x}}_{k|k-1}\hat{\mathbf{x}}^{T}_{k|k-1} + \mathbf{Q}_{k-1} \tag{14} \]

Measurement-Update

  • Factorize \[ \mathbf{P}_{k|k-1} = S_{k|k-1}S_{k|k-1}^{T} \tag{15} \]
  • Evaluate the cubature points: (\(i=1, 2,\ldots m, \text{ where } m=2n_{x}\)) \[ X_{i, k|k-1} = S_{k|k-1}\xi_{i} + \hat{\mathbf{x}}_{k|k-1} \tag{16} \]
  • Evaluate the propagated cubature points \((i=1,2,\ldots m)\): \[ Z_{i, k|k-1} = \mathbf{h}(X_{i,k|k-1}, \mathbf{u}_{k}) \tag{17} \]
  • Estimate the predicted measurement \[ \hat{\mathbf{z}}_{k|k-1} = \frac{1}{m}\sum_{i=1}^{m}Z_{i, k|k-1} \tag{18} \]
  • Estimate the innovations covariance matrix \[ \mathbf{P}_{zz, k|k-1} = \frac{1}{m}\sum_{i=1}^{m}Z_{i, k|k-1}Z^{T}_{i, k|k-1} - \hat{\mathbf{z}}_{k|k-1}\hat{\mathbf{z}}^{T}_{k|k-1} + \mathbf{R}_{k} \tag{19} \]

Page 6

Measurement-Update (Continued)

  • Estimate the cross-covariance matrix \[ \mathbf{P}_{xz, k|k-1} = \frac{1}{m}\sum_{i=1}^{m}X_{i, k|k-1}Z^{T}_{i, k|k-1} - \hat{\mathbf{x}}_{k|k-1}\hat{\mathbf{z}}^{T}_{k|k-1} \tag{20} \]
  • Estimate the cubature Kalman Gain \[ W_{k} = \mathbf{P}_{xz, k|k-1}\mathbf{P}^{-1}_{zz, k|k-1} \tag{21} \]
  • Estimate the updated state \[ \hat{\mathbf{x}}_{k|k} = \hat{\mathbf{x}}_{k|k-1} + W_{k}(\mathbf{z}_{k} - \hat{\mathbf{z}}_{k|k-1}) \tag{22} \]
  • Estimate the corresponding posterior error covariance matrix \[ \mathbf{P}_{k|k} = \mathbf{P}_{k|k-1} - W_{k}P_{zz,k|k-1}W^{T}_{k} \tag{23} \]
