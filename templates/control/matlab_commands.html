Matlab Commands for Control Theory

Time-Domain Function Description
step Plot step response of a dynamic system.
impulse Plot impulse response of a dynamic system.
initial Plot system response to initial conditions.
lsim Simulate response of LTI system to arbitrary input.
stepinfo Retrieve step response characteristics (rise time, settling time, etc.).
dcgain Compute steady-state (DC) gain of the system.
ltiview Open LTI Viewer for analyzing time and frequency responses.
ss Create a state-space representation of a system.
Frequency-Domain Function Description
bode Plot the Bode magnitude and phase plot.
nyquist Generate Nyquist plot of the system.
nichols Generate Nichols plot of the system.
margin Calculate gain margin, phase margin, and crossover frequencies.
freqresp Compute system frequency response at specific frequencies.
sigma Plot singular value frequency response.
bandwidth Compute bandwidth of the system.
tf Create a transfer function representation.
Additional Function Description
rlocus Plot root locus of a system.
feedback Create closed-loop system from open-loop transfer function.
lqr Design Linear Quadratic Regulator (LQR) controller.
kalman Design Kalman state estimator.
place Place poles for state-feedback control.
pidtune Automatically tune PID controller gains.
controlSystemDesigner Open GUI for interactive control system design.
linearSystemAnalyzer Open GUI for analyzing LTI systems.
hinfsyn Design H-infinity optimal controller.
