LQE for DC Motor Control

LQR Control Law Derivation

Objective

The goal of Linear Quadratic Regulator (LQR) design is to determine a state feedback control law:

u = -K * x

where K is the feedback gain matrix that minimizes a quadratic cost function.

Cost Function

The quadratic cost function is given by:

J = ∫ (xTQx + uTRu) dt
  • Q: A positive semi-definite matrix penalizing state deviations.
  • R: A positive definite matrix penalizing control effort.

System Dynamics

The linear time-invariant system is described by:

dx/dt = Ax + Bu

where:

  • x: State vector.
  • A: State matrix.
  • B: Input matrix.
  • u: Control input.

Solution Approach

The LQR problem is solved using the following steps:

  1. Define the cost function: Formulate the quadratic cost function based on system objectives.
  2. Compute the Riccati Equation: Solve the Continuous Algebraic Riccati Equation (CARE):
    3. ATP + PA - PBR-1BTP + Q = 0
  3. Determine Feedback Gain: Compute the optimal feedback gain as:
    4. K = R-1BTP

Key Insights

  • The feedback gain K minimizes the cost function J.
  • The Riccati equation ensures stability and optimality of the control law.

Advantages of LQR

  • Provides a systematic method for designing optimal controllers.
  • Balances state regulation and control effort.
  • Ensures robustness for small perturbations in the system.

MATLAB Implementation

The MATLAB lqr function directly computes the feedback gain K. Example:

% Define system matrices
A = [0 1; -2 -3];
B = [0; 1];
Q = eye(2);  % State penalty matrix
R = 1;       % Control effort penalty

% Compute LQR gain
K = lqr(A, B, Q, R);

% Display result
disp('LQR Gain:');
disp(K);

The computed K can then be used in the control law:

u = -K * x

Dynamic State-Space Model (DSSM) of a DC Motor

Introduction

The Dynamic State-Space Model (DSSM) is a mathematical framework used to describe the behavior of a DC motor. It captures the dynamics of the motor using state variables, input signals, and output signals. This representation is widely used for control system design and analysis.

State-Space Representation

The state-space equations for a DC motor are given by:


    dx/dt = A * x + B * u
    y = C * x + D * u

State Variables

  • x1: Armature current (ia).
  • x2: Angular velocity of the motor shaft (ω).

Input and Output

  • Input (u): Voltage applied to the motor (Va).
  • Output (y): Angular velocity (ω).

Matrices

The state-space matrices for the DC motor are:

  • A: State matrix, which defines the system dynamics.
  • B: Input matrix, which defines how the input affects the states.
  • C: Output matrix, which maps the states to the output.
  • D: Direct transmission matrix, which maps input directly to the output.

    A = [[-R/L    -K/L],
         [K/J  -B/J]]

    B = [[1/L],
         [0]]

    C = [[0  1]]

    D = [[0]]

Physical Parameters

The following parameters are used in the model:

  • R: Armature resistance (Ohms).
  • L: Armature inductance (Henries).
  • K: Back EMF constant (Volt-sec/rad).
  • J: Moment of inertia of the rotor (kg-m2).
  • B: Viscous friction coefficient (N-m-s).

Applications

The DSSM of a DC motor is used in:

  • Control system design (e.g., PID, LQR).
  • Simulation of motor dynamics.
  • Observer design (e.g., Kalman filter).
  • Hardware implementation and testing.

Conclusion

The state-space representation provides a compact and structured way to analyze and control the behavior of a DC motor. By leveraging this model, engineers can design advanced control algorithms and improve motor performance in various applications.
